# HQ Données Ouvertes

## Installation

 - Install [swiss-army-knife-card](https://github.com/AmoebeLabs/swiss-army-knife-card) card from Hacs.
    - You must use the last dev release to use this card.
 - Copy the templates
 - Restart Hone Assistant
